# <img src="./VALR.Net/icon/icon.svg" alt="Icon" width="64" height="64"> VALR.Net

VALR.Net is a .Net wrapper for the VALR API as described on [VALR](https://docs.valr.com/).

This API wrapper is based on the pattern used by JKorf in his Bitfinex.Net project [Bitfinex.Net](https://github.com/JKorf/Bitfinex.Net).

It is planned to includes all features the API provides using clear and readable C# objects including 
* Reading market info
* Placing and managing orders
* Reading balances and funds
* Live updates using the websocket

Additionally it adds some convenience features like:
* Configurable rate limiting
* Autmatic logging

**If you think something is broken, something is missing or have any questions, please open an [Issue](https://github.com/lilpaul/VALR.Net/issues)**
